VitalWear Desktop App

🔌 Connection Status: Server Required

The application needs XAMPP server to run properly.

📋 Setup Instructions:

  1. Install XAMPP if not already installed
  2. Start XAMPP Control Panel
  3. Start Apache Server (click "Start" button)
  4. Start MySQL (click "Start" button)
  5. Restart this application

🔍 Troubleshooting:

Port 80 already in use? Change Apache port to 8080

Apache won't start? Check for conflicting services

Still issues? Try running as Administrator
